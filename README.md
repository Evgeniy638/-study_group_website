# -study_group_website

## Как запустить локально (node.js должен быть уставлен)

1) скопируйте репозиторий и перейдите в ветку `develop`

2) установите `node modules` следующей командой
### `npm install`

3) запустите базу данных локально
### `npm run dbstart`

4) запустите проект
### `npm start`
